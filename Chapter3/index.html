Array Review Questions

1. What is the difference between using "quotation marks" versus 'apostrophes' in JavaScript?

Functionally there is no difference between using " or ' in JavaScript. However, if you want quotation marks as part of the string you must use a pair of apostrophes wrapping around the quotation marks.

2. What is the key difference between the innerHTML property and the textContent property?

With innerHTML you can include any HTML code as part of the assignment and the browser will render its style correctly. However, textContent only modifies the text. If you include HTML code using the textContent property you'll see the code reflected in the browser's rendering.

3. For a reference to a month, why might it make more sense to reference an array value vs. a variable?

For an array of months, it's easier to assign and reference each month numerically as we might think of them in real life. Such as the ninth month of the year is September, and accounting for array indexing in our project that is months[8]. With a variable we can only store one value, which may work well if we we're referencing or assigning a variable of the current month or something.

Assuming showTotal is declared function showTotal(tax, total) { } the function call is showTotal(2, 10);

1. What is the benefit of using literal notation over constructor notation to create an object?

If you need more control over an object that will be created, you can use literal notation to create the object and its properties and methods.

2. What is the benefit of using constructor notation over literal notation to create an object

You can create a template of an object using a function which may or may not create objects based on your needs. It can also be used to easily create multiple instanced objects based on that template.

The window object.

The containing object for that method.
